## What is Saros?

**Saros = SaaS + OS.** Not a gym management tool. An AI store co-founder.

| Component | Role |
|:--|:--|
| **Momo** | 门店企业大脑 · 创始人姓氏"莫"的数字分身 · 7年门店运营认知编码 |
| **SaaS Stack** | 执行手脚 · POS/排课/CRM/进销存/会员管理/开源部署 |

Momo knows how to run a gym — when to schedule classes, follow up with members, adjust pricing. The SaaS stack is her limbs. Together, they form Saros: a digital co-founder that runs the store autonomously.

**Traditional SaaS:** "Here are tools. You figure it out."
**Saros:** "Here is a store manager who knows how to use those tools."

```bash

cd zwiserfit-ai-store-manager
npm install -g openclaw
openclaw init --profile momo-store
openclaw agent start momo --capabilities store-ops,member-mgmt,daily-report
```

# Momo SaaS Stack

> 🟢 MIT | Momo（门店大脑） + SaaS 栈源码

本目录存放 Saros 的 Momo 门店运营操作系统及配套 SaaS 工具栈源码。

**主仓库：** [ZWISERFIT/zwiserfit-ai-store-manager](https://github.com/ZWISERFIT/zwiserfit-ai-store-manager)

## 组件

- **Momo Agent OS** — 门店运营大脑，7年门店运营认知编码
- **POS Module** — 收银/支付集成
- **CRM Module** — 会员管理/跟进
- **Scheduling Module** — 排课/预约
- **Inventory Module** — 进销存管理
- **Report Module** — 日报/周报/月报自动生成

## 技术栈

- OpenClaw Agent Framework
- Hermes Agent Framework (双框架冗余)
- Node.js / TypeScript

## The Set-Up

9 specialized AI agents, each with:
- An identity (SOUL.md)
- A domain (finance, tech, community, compliance, growth, operations, assets, governance)
- Independent tool access (code, search, messaging, sensors)
- Cross-validation responsibility (each agent audits others' outputs)
- A constitutional framework that governs behavior

They don't wait for instructions. When an event happens, the relevant agent responds autonomously — then reports after action.

## Why This Exists

ZWF is a physical business first — 7 gyms, real hardware, real members, real P&L. The AI agent collective exists because running a distributed physical business requires:

1. **24/7 monitoring** — hardware doesn't sleep
2. **Cross-domain coordination** — finance affects ops affects compliance affects brand
3. **Autonomous response** — by the time a human reads the alert, the agent has already acted

We didn't build agents because agents are cool. We built them because physical infrastructure demands it.

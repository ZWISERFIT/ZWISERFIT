## P2 · 问题：40亿人的物理行为没有资产化路径

**核心论述：**

全球40亿人每天产生运动行为——但这些行为被平台捕获、被设备记录、被免费货币化。产生行为的人不是行为的拥有者。他们只是"用户"。

**三个支撑点：**

1. **行为被捕获但不被拥有。** Apple Watch知道你今天跑了5公里——但Apple拥有那个数据。你得到了什么？一条通知。"恭喜你完成了今日目标。"行为数据的商业价值完全被平台提取——零归属于行为生产者。

2. **Multicoin Capital的ILM框架缺了一行。** 2026年3月，Multicoin发表"Internet Labor Markets"论文——定义了数字劳动的论贡献者拥有市场。但矩阵里全是数字劳动（地图测绘/配送/漏洞赏金/模型评估）。物理行为——40亿人每天都在做的最通用的"劳动"——仍然是ILM方程中的零。

3. **保险/健康行业需要规模化的行为数据——但没有采集管道。** 行业调研显示保险公司愿意为行为数据支付¥50-250/人/年——但没有规模化、可验证、隐私保护的采集基础设施。健身房记录数据→平台拥有→用户看不到→保险公司买不到。管道断裂在"谁拥有"的环节。

**市场规模锚点（a16z 7 Charts 2026）：**
- DePIN赛道：$3.5T 潜在市场
- 健康数据市场：$34B → 预计2030年达$100B+
- 可穿戴设备：5亿+活跃设备，行为数据零用户主权

## P3 · 方案：PoPB协议——Model 5: Physical-Native RWA

**核心论述：**

ZWISERFIT建立了一个新的RWA品类——**Model 5: Physical-Native RWA**。

Multicoin Capital的"RWAs Are Just Built Different"(2026.03.19)定义了四种RWA上链模型：Synthetic Derivatives(合成衍生品)、Wrapped Assets(托管包装)、Collateralized Borrowing(抵押借贷)、Primary Onchain Issuance(原生链上发行)。四种模型共享一个隐藏假设：**资产先于证明存在。**

如果证明本身就是资产呢？

**Model 5: Physical-Native RWA**的运作方式：
1. 人在传感器环境中完成物理行为（如100个深蹲）
2. 15维传感器融合实时捕获
3. 硬件安全芯片（TEE）用出厂烧录的Ed25519密钥签名
4. 行为证明上链——证明=资产，不需要tokenization
5. 用户通过DID持有自己的行为数据主权

**三个支撑点：**

1. **不需要tokenization。** 行为证明本身就是资产。不像股票需要"代币化"——行为证明原生存在于链上。没有"底层资产"需要托管。没有"价格预言机"需要喂价。

2. **硬件信任根=不可伪造。** 每一个行为证明必须包含硬件TEE签名。纯软件实现=数学上不可能产生有效证明。这不是"我们相信你不会造假"——这是密码学安全边界。

3. **隐私保护=架构设计，不是承诺。** 原始传感器数据通过MPC分片存储于独立方。ZWF作为协议开发者无法访问任何用户的原始数据。这不是隐私政策——这是技术架构。

## P4 · 产品与技术：PoPB五层协议栈

**核心论述：**

PoPB（Proof of Physical Behavior）是一个开放、硬件锚定的五层协议，标准化人类物理行为的采集、验证、传输和资产化。

```
Layer 5: APPLICATION — DID验证·保险精算·行为资产交易
Layer 4: VERIFICATION — 分布式验证网络·共识·惩罚(Slashing)
Layer 3: PRIVACY — MPC数据分片·零知识证明·DID主权
Layer 2: DATA CAPTURE — 15维传感器融合·信号处理管线
Layer 1: HARDWARE TRUST ROOT 🔒 — 安全芯片·TEE·设备认证链
```

**TCP/IP类比：**
```
TCP/IP : 互联网通信 :: PoPB : 人类物理行为数据
```

TCP/IP标准化了异构网络之间的通信协议。PoPB标准化了人类物理行为的采集、证明、传输和资产化——跨硬件、跨司法管辖区、跨应用领域。

**三个支撑点：**

1. **Layer 1是数学级壁垒。** 每台门禁传感器出厂时烧录唯一Ed25519密钥。硬件TEE隔离执行环境。安全启动链验证固件完整性。纯软件实现可以写出符合JSON格式的证明——但无法生成有效的device_attestation字段。这是密码学边界，不是策略边界。

2. **Layer 2是复杂度壁垒。** 15维传感器融合（IMU/生物阻抗/压力矩阵/ToF深度/红外热成像）在200Hz+采样率下同步。AI可以生成一个深蹲视频。AI无法生成15个传感器在物理约束一致性下的同步数据流。攻击面=不可能。

3. **Layer 3是结构性隐私。** 用户通过DID持有自己的数据。MPC确保没有任何单一方看到完整原始数据。ZK证明允许"证明行为发生"而不暴露"谁做了"。保险公司可以验证"此人完成100个标准深蹲"而不知此人是谁——除非此人主动授权。

## P5 · 单店财务模型：东莞万江验证版

**核心论述：**

ZWISERFIT在东莞万江（城乡结合部·中国健身渗透率最低的地区之一）运营了7年。这是在**最差条件下测试了模型的下限**——然后证明了它work。

**关键数字：**

| 指标 | 数值 | 说明 |
|------|:----:|------|
| 单店面积 | ~800㎡ | 一线城市同规模成本的25-40% |
| 单店建设成本 | ~¥200万 | 含四类折旧（硬件/装修/设备/IT） |
| 7年持续运营 | ✅ 已验证 | 经历疫情·长期亏损·仍保持运营 |
| 行为捕获率 | 100% | 每进入一次门禁=一次自动验证 |
| 会员留存 | 7年趋势向上 | 2026年东莞健身氛围首次醒过来 |

**为什么选择东莞万江（不是弱点·是最硬论证）：**
- 选了渗透率最低的地方→客户获取最难→成本最高→但7年后仍然活着
- 证明：即使在最难的地方，节点也能存活+100%验证捕获率+7年趋势线向上
- 复制到渗透率更高的地方→所有变量改善，没有变量恶化→非对称上行空间

**复制成本：** 标准化SOP+AI Agent运营=单店复制<¥200万。AI Agent替代了传统健身房的店长/教练/SaaS费用。人才瓶颈已被消除。

| 类型 | 代表 | 核心能力 | 为什么不是直接竞争 | 防御策略 |

| **数字健身房SaaS** | eGym, GeniusAI | 软件+简单传感器 | 无硬件信任根·无链上资产化·无协议层 | 品类差异=不是同一赛道 |
| **可穿戴巨头** | Apple, Google | 十亿级设备·AI能力·无限资金 | 单点传感器(手腕)·不开源·无链上验证 | 协议开放先发+多维度传感器=质差 |
| **健身连锁** | 乐刻, Keep | 门店网络·用户量 | 无硬件信任根·无链上资产化 | 可能的合作方·非竞争者 |
| **DePIN协议** | Helium, Hivemapper | 链上验证·代币经济 | 验证"设备在哪"·不验证"人体做了什么" | PoPW≠PoPB·互补非竞争 |

**Big Tech（Apple/Google）是唯一真实的长期威胁：**
- 防御1：协议开放先发。先定义标准的那个实体拥有品类定义权。
- 防御2：15维传感器融合=质差，非量差。手腕式传感器≠全身门禁传感器。
- 防御3：开源+隐私保护架构=Apple的围墙花园无法兼容。

## P8 · 代币经济学：Arcade Token模型

**核心论述（基于a16z crypto "Arcade Tokens" 2026年5月论文）：**

a16z crypto在2026年5月发表"Arcade Tokens"论文——定义了一种"在证券法之外的token类别"：Arcade Token的经济价值来自消费效用，而非投资预期。ZWF的行为资产token完全符合Arcade Token的定义。

**行为资产模型（BAT: Behavior Asset Token）：**
- 用户完成验证行为→自动铸造BAT→DID钱包
- BAT代表该次行为的不可伪造的链上证明
- BAT可在行为交易市场上交易（类似Pop Mart限定款溢价模型）
- "连续30天每天练腿""东莞万江第一个完成1000次深蹲的人"=不可伪造的身体成就=天然稀缺性

**两条收入腿：**
- **B端：** 保险公司/健康平台购买匿名行为数据→¥50-250/人/年（线性增长）
- **C端：** 行为稀缺性交易市场→溢价无限定（指数级潜力）

**代币不是证券：** a16z Arcade Token框架+PoPB硬件信任根=消费效用型token，非证券。
